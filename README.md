## 📖 简介

BalatroQt 是一款回合制的扑克主题 Roguelike 构筑游戏。每一局中，你从自己的牌库抽取手牌，
挑选若干张组成扑克牌型出牌得分；通过击败一关又一关的盲注，进入商店购买小丑牌、消耗牌与各类道具，
逐步把一副平平无奇的扑克，改造成能打出天文数字的得分机器。

项目使用 **Qt 6 + C++17** 从零实现了完整的游戏循环、计分系统、道具体系与大量视觉特效，
UI、动画与音效均为原生实现，力求流畅细腻的手感。

---

## ✨ 核心特色

- 🎯 **完整的计分系统** —— 13 种扑克牌型，筹码（Chips）× 倍率（Mult）逐张结算，支持 `×倍率` 叠乘。
- 🤡 **30+ 张小丑牌** —— 常驻在场提供各式加成与联动效果，是构筑套路的核心。
- 🔮 **丰富的消耗牌** —— 24 张塔罗牌、12 张星球牌、18 张幻灵牌，用于强化牌面、升级牌型或改造牌库。
- 💎 **卡牌改造体系** —— 增强、版本（Foil/全息/多彩/负片）、蜡封（金/红/蓝/紫）自由组合。
- 👑 **多层盲注挑战** —— 小盲注、大盲注与 20+ 种机制各异的 Boss 盲注。
- 🛒 **商店与开包** —— 重随、购买、道具槽管理，多种补充包开启体验。
- 🎨 **GPU 特效与动画** —— 大量自定义着色器与物理弹簧动画，还原细腻的卡牌质感。

## 🛠️ 技术架构

| 层 | 说明 |
| :-- | :-- |
| **框架** | Qt 6（Core / Widgets / OpenGL / OpenGLWidgets / Multimedia） |
| **语言** | C++17 |
| **构建** | CMake（≥ 3.19），资源经 `qt_add_resources` 打包进可执行文件 |
| **渲染** | `QGraphicsScene` / `QGraphicsView` 承载卡牌场景，OpenGL 视口 + 离屏 FBO 运行自定义着色器 |
| **游戏逻辑** | `gamestate` 统一驱动回合流程与计分，`handevaluator` 负责牌型判定，道具体系模块化拆分 |
| **音频** | `audiomanager` 统一调度音效与音乐，支持音高 / 音量调制 |

代码按 **卡牌（card）/ 游戏逻辑（game）/ 界面（ui）/ 工具（utils）/ 音频（audio）** 分模块组织，职责清晰。

## 📦 构建与运行

> 环境要求：Qt 6.5 或更高版本、支持 C++17 的编译器（MinGW / MSVC）、CMake ≥ 3.19。

```bash
# 进入项目源码目录
cd BalatroQt

# 配置（请将路径替换为本机 Qt 安装目录）
cmake -S . -B build -DCMAKE_PREFIX_PATH="D:/Qt/6.10.0/mingw_64"

# 编译
cmake --build build --target BalatroQt

# 运行
./build/BalatroQt
```

也可直接用 **Qt Creator** 打开 `BalatroQt/CMakeLists.txt`，选择套件后一键构建运行。

## 🕹️ 操作说明

| 操作 | 说明 |
| :-- | :-- |
| **点击手牌** | 选中 / 取消选中，最多选 5 张 |
| **出牌** | 将选中的牌作为牌型打出计分 |
| **弃牌** | 弃掉选中的牌并补抽，每回合次数有限 |
| **拖拽手牌** | 调整手牌顺序 |
| **悬浮卡牌** | 查看该牌的详细信息面板 |
