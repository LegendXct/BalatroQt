# BalatroQt 路演录屏脚本 v2（演示模式版，4:00）

> 目标：让完全不懂《小丑牌》的助教在 4 分钟内（1）听懂玩法，（2）看清我们在原版规则之上的算法/系统级创新。
> 原则：弱化 UI、强调**算法与功能实现**，每一段都给一句"代码层面在做什么"的台词。
> **本脚本依赖游戏内置的演示模式**——开关在 `设置 → 演示模式`，开启后每次"开始新的一局"按下面剧本固定运行。

---

## 录屏前准备（开拍前 3 分钟做完）

1. **构建**：`cmake --build build/Desktop_Qt_6_11_0_MinGW_64_bit-Debug` 已通过；窗口为全屏。
2. **进入设置** → **演示模式打开**。回到主菜单。
3. **窗口**：录 1920×1080；OBS 启动；音量主 30%、音效 60%、倍速 1×（演示中会改）。
4. **演示模式开启时的固定剧本**：
   - 小盲注开局手牌（最前 8 张固定）：`♠5 ♦5 ♥6 ♣7 ♦8 ♦9 ♣10 ♦2`
   - 大盲注开局手牌：`♥9 ♥J ♥Q ♥K ♥A ♣4 ♦4 ♠7`
   - Boss(支柱)开局手牌：`♠6 ♠7(玻璃) ♠8 ♠9 ♠10 ♣2 ♥3 ♥4`
   - 第一商店：主货架 = 蓝图 + 头脑风暴；重摇后 = DNA + 特里布莱；voucher = Grabber（+1 出牌）；包 = 天体包(必含木星) + 奥秘包(必含正义)
   - 第二商店：摆设位（演示设置后不买）
   - Boss = **The Pillar 支柱**（本 Ante 已出过的牌全部禁用）

5. **金币核算**（脚本保证够花）：
   - 起步 $104 → 小盲奖励 $3 + 剩 2 出牌 $2 + 利息 $5 = 进商店 **$114**
   - 蓝图$10 + 头脑风暴$10 + 重摇$5 + DNA$8 + 特里布莱$20 + Grabber$10 + 天体包$4 + 奥秘包$4 = **$71**
   - 剩 $43 进大盲，宽裕。

## 演示模式相关代码索引（讲解时可点到）

- 开关 & 状态：`src/game/demoscript.h/.cpp` —— 4 个静态计数器 + 6 个 hook 函数。
- 牌堆重排：`Deck::reset()` 末尾调 `reorderDeckForNextBlind` —— 普通 `std::find_if` + `iter_swap`。
- 商店货架：`Shop::rerollShopOnly()` / `Shop::roll()` 顶部检查 `DemoScript::active()`，命中则用脚本货架。
- 包内容：`generatePackContent` 顶部 hook，覆盖天体/奥秘包。
- Boss：`randomBossEffect` 顶部 hook，第一 Ante 强制 The Pillar。
- 鼠标光标修复：`mainwindow.cpp` 给 `mView` 与 `mView->viewport()` 显式 `setCursor(Qt::ArrowCursor)`。
