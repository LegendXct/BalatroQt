# BalatroQt 路演录屏脚本 v4（4:00 / 演示模式）

> 目标：让完全不懂《小丑牌》的助教在 4 分钟内（1）听懂玩法，（2）看清算法/系统级创新。
> 旁白原则：**只讲功能不讲代码**——所有函数 / 类名挪到段末"代码定位（后期叠图）"里，后期把名字 / 路径 / 行号叠到对应画面上即可。
> **依赖游戏内置演示模式**：开关在 `设置 → 演示模式`。

## 录屏前准备（开拍前 3 分钟）

1. 构建已通过；启动 `BalatroQt.exe`，直接进主菜单（不再有盲注页闪屏）。
2. **设置** → 勾选"演示模式" → 返回主菜单。倍速保持 1×（演示中再切）。音量主 30%、音效 60%。

## 本轮顺带修的 bug

1. **鼠标光标全屏不见** —— `mainwindow.cpp` 给 `mView` 与 `mView->viewport()` 都 `setCursor(Qt::ArrowCursor)`。
2. **进度条彩色填充戳出药丸框** —— 给 `mScoreProgressBar` 加 antialias QBitmap mask，eventFilter 在 Resize/Show 时同步重建。
3. **启动时盲注选择页闪屏** —— `showMainMenuOverlay()` 从 `singleShot(0)` 提到构造体同步调用；`resizeEvent` 补 `mMainMenuOverlay` 撑满 centralWidget。
4. **倍速影响 UI 过渡 / 消耗品动画** —— `scaledDelay()` 加守卫，`!mScoringInProgress` 时直接返回原值，倍速只缩短计分链。
5. **翻面牌悬浮泄露信息** —— `showCardHoverTooltip` 检查 `c.faceUp`，false 时只显示三个问号。
6. **演示模式商店优惠券错成 Grabber** —— `DemoScript::scriptedVoucherOffers` 改回 Overstock。
7. **演示模式 Mega Buffoon 被覆盖成 Normal** —— `gamestate.cpp` 第一商店硬塞 `setBoosterOfferPack(0, Buffoon, Normal)` 的分支加 `!DemoScript::active()` 跳过。
