# BalatroQt 路演演示脚本 V2（5:00 / 现场双人 / 演示模式）

> 形式：**操作员**（控制游戏）+ **主讲**（台词），现场实机演示 5 分钟。
> 主线：**只从第一小盲打到第一底注 Boss 通关**。
> 设计原则：基础玩法（筹码×倍率、商店、盲注）一句话带过；**镜头时间全部让给程设特色**——
> 定制牌组（老师头像 JQKA）、队列牌组（三选一）、四张程设卡（运算符重载 / 类模板 / 迭代器 / 浅拷贝），
> 同时保留占卜、最佳出牌、进度条三个自研创新。
> ⚠ 本剧本需要按文末「附录 A：DemoScript V2 改造清单」更新演示模式固定剧本后才能照表执行。

## 开演前准备（候场时完成）

## 附录 A：DemoScript V2 改造清单（照此更新演示模式后剧本方可执行）

> 现有 `src/game/demoscript.h/.cpp` 仍是 V1 录屏剧本（蓝图/头脑风暴/特里布莱/The Mark）。V2 需要：

1. **三个盲注的发牌序**改为本脚本「演示模式固定剧本」表中的顺序（队列牌组下发牌序 = 手牌序，`Deck::reset()` 末尾的 `reorderDeckForNextBlind` 逐张钉死，含小盲两次补牌 `♦J ♥J` / `♣K ♠4 ♥2`）。
2. **Boss 局发牌**需按 **uid** 找回两张动态牌：大盲中被浅拷贝/迭代过的两张 ♥10（原 ♥9 与其拷贝），登记 uid 后在 reorder 时优先入列；同时保证 `♥A ♥8 ♥5` 在窗口内。
3. **商店接管**：`scriptedShopOffers` visit 1 = [运算符重载, 类模板]；visit 2 = [木星 + 两张任意普通小丑]。
4. **包接管**：`scriptedBoosterOffers` visit 1 = Mega Arcana；`scriptedPackContent` Mega Arcana 分支 = 迭代器 + 浅拷贝 + 3 张陪衬塔罗（**无灵魂**）。
5. **Boss 固定**：`randomBossEffect` 演示分支 The Mark → **TheWall**。
6. **起步金币**沿用 $104；V1 剧本专用 hook（灵魂→特里布莱、公牛挂多彩等）保留不影响，但 visit 1 硬塞 Buffoon 的旁路注意不要复活。
7. **演示模式 × 队列牌组兼容性**过一遍：选牌组在 startGame 之前注入，确认 demo reorder 在队列发牌路径上同样生效。

## 附录 D：操作员备忘

1. **队列牌组全程禁拖拽、禁排序**——别习惯性整理手牌；选牌只点击。
2. 浅拷贝方向是**靠左拷贝靠右**（位置关系，与点击先后无关）；剧本已保证 ♦3 在 ♥9 左边。
3. 迭代器那一步**只选拷贝出来的那张**（位置 2），同步效果才有戏剧性。
4. 倍速切换点只有两处：**进大盲后 0.5×**、**进第二商店后 1×**。
5. hover 节奏：台词说到哪张卡，鼠标停在哪张卡上，停足 1 秒再动。
